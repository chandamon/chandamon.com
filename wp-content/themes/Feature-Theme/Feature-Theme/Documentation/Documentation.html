03. UPLOADING & ACTIVATING THE PLUGINS -Back to Top

Inside the plugins folder you will see plugins named "wp-pagenavi" and "limit-post" upload that pluing to the "wp-content/plugins" folder in your Wordpress directory online in your hosting account via FTP access.
Now if that plugin, has been uploaded successfully to your server, than access the plugin panel in your Wordpress Administration Panels and check out through the list of plugins to find out your new plugin which you just uploaded. Once you find it than click on the "Activate link" to turn that plugin on.

04. UPLOADING THE THEME -Back to Top

Inside the ZIP folder you will see a folder named "feature" upload the "feature" folder to the "wp-content/themes" in your Wordpress directory online in your hosting account via FTP access.

05. ACTIVATING THE THEME -Back to Top

Now if you have uploaded the theme and the plugin to the correct directory, so activating your new theme is click away.
1. Login in to the Wordpress Administration Panel.
2. Select the Appearance Panel, than click on Themes.
3. Now you will see your theme here, click on the theme title (or theme screenshot) in order to activate it.

06. CHANGING THE LOGO IMAGE -Back to Top

You can change the theme images, using the feature theme control panel. Make sure you use the correct image URL paths.
So in order to replace the logo, image in control panel, you must Enter the URL of your custom image like below: also avoid signs like, 
?
=
etc, also avoid any extra empty spaces.
See the examples below:-
WRONG : www.yoursite.com/images/mylogo.jpg
WRONG : http://yoursite.com/imgaes/mylogo.jpg
WRONG : www.yoursite.com/?images=mylogo.jpg
CORRECT : http://www.yoursite.com/images/mylogo.jpg

07. POST THUMBNAILS IMAGES -Back to Top

No Custom Field for Thumbnails, you do not require to add the Wordpress custom field for the posts thumbnails, we have coded our own custom field, and you will find this below the writing posts window.
To add post thumbnail images, see below the writing a post window in Wordpress, you will have the "Feature Theme Post Thumbnail Setting URL Box", so there in the box simply paste the full URL of your image, like http://www.yoursite.com/images/cake.jpg
OR if you are using Wordpress media library, than don't include the http://www.sitename.com
use it like example below:
/wp-content/uploads/2010/01/example_image.jpg
and save the post, the image will be re-size automatically to fit in the theme post thumbnails.

8. SETTING YOUR EMAIL FOR CONACT PAGE -Back to Top

The Feature Theme has a custom contact page, so for receiving emails, you need to set up your email address. It's easy, in the theme folder, open up contact.php file and than go to line number 51 and change the mail@gmail.com with your email address. and than save the file.
For adding the contact page, go to add new page and create a page, and give it any name for example "contact" do not write any thing on the notepad window,
than on the right pages attributes you will see a drop down menu below the "Template" than select contact and publihs the page,
than you need to add the pages widget, than the contact page will appear there. Or create a custom menu and add contact page into it, than add the custom menu widget.

9. ADDIN A CUSTOM MENU -Back to Top

To create a custom menu, give it a name and click Create Menu. Then choose items like pages, categories or custom links from the left column to add to this menu.
After you have added your items, drag and drop to put them in the order you want. You can also click each item to reveal additional configuration options.
When you have finished building your custom menu, make sure you click the Save Menu button.
Watch the video tutorial, you must have internet connection.

than on the right pages attributes you will see a drop down menu below the "Template" than select any layout and publihs the page,
than you need to add the pages widget, than the layout page will appear there. Or create a custom menu and add layout page into it, than add the custom menu widget.
